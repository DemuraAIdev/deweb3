Semua orang pasti punya portofolio, baik itu portofolio untuk pekerjaan, portofolio untuk sekolah, portofolio untuk kuliah, portofolio untuk keperluan lainnya. Portofolio ini biasanya berupa kumpulan karya yang telah kita buat. Kali ini saya akan membahas tentang portofolio yang saya buat untuk keperluan pribadi.

## Mengapa saya membuat portofolio?

Saya membuat portofolio ini dengan tujuan untuk mempermudah orang lain melihat karya-karya yang telah saya buat. Selain itu, saya juga ingin memperkenalkan diri saya kepada orang lain. Karena dengan portofolio ini, orang lain dapat melihat karya-karya saya dan orang lain juga dapat mengetahui siapa saya.

## Bagaimana cara saya membuat portofolio ini?

Saya membuat portofolio ini dengan menggunakan [Next.js](https://nextjs.org/). Next.js adalah framework React yang dapat digunakan untuk membuat website. Next.js juga memiliki fitur static site generation yang dapat membuat website menjadi lebih cepat. Selain itu, Next.js juga memiliki fitur server side rendering yang dapat membuat website menjadi lebih cepat.

Saya juga mengambil inspirasi dari beberapa website seperti [Lee Robinson](https://leerob.io/), [Thvu](https://www.thvu.dev/), dan lain-lain.

## Apa yang saya gunakan untuk membuat portofolio ini?

Saya menggunakan beberapa teknologi untuk membuat portofolio ini, yaitu:
- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [MDX](https://mdxjs.com/)
- [Vercel](https://vercel.com/)
- [Github](https://github.com/)

## Apa yang harus saya isi di portofolio ini?

Saya menyarankan untuk mengisi portofolio ini dengan karya-karya yang telah anda buat. Anda juga dapat mengisi portofolio ini dengan informasi tentang diri anda. Anda juga dapat mengisi portofolio ini dengan informasi tentang pekerjaan anda.
